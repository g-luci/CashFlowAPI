# CashFlowAPI - Projeto de prática

> [!IMPORTANT]
> Este projeto contempla o conteúdo apresentado pela Rocketseat e serve como um projeto de aprendizagem e prática.

Este projeto tem como objetivo, criar uma API completa, utilizando Domain Drive Design, injeção de dependência, async, await, testes de unidade e mais. O projeto será um manager de despesas pessoais, com funcionalidades como adicionar, atualizar, deletar e listar despesas, além de gerar Excel e PDF.
